# Skill: draft

Write a first draft from an outline, brief, or set of notes.

## Input

- An outline (preferred) or brief
- Optional: tone, audience, length target, format constraints

## Output

A complete first draft in the requested format. Default to markdown.

## Instructions

1. Follow the outline structure if provided; otherwise derive structure from the brief.
2. Write for the stated audience. If not specified, assume an informed non-specialist.
3. Prioritize clarity and flow over polish — this is a first draft.
4. Use concrete examples and specifics; avoid vague generalities.
5. Mark genuinely uncertain or placeholder content with `[TODO: ...]`.
6. After the draft, add a **Draft notes** block:
   - Word count
   - Sections that need the most work
   - Any assumptions made

## Tone defaults

- Professional but not stiff
- Active voice
- Second person ("you") for instructional content; third person for analysis
